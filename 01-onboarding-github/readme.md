# 📁 Módulo 01: Onboarding e Colaboração com Git/GitHub

Este diretório contém os registros do primeiro contato prático com o fluxo de trabalho colaborativo no ecossistema Open Source durante o Bootcamp Almaviva.

## 🎯 Objetivo do Módulo
O foco principal foi a imersão em conceitos fundamentais de controle de versão, abrangendo:
- Fluxo de **Fork** e **Pull Request**.
- Gestão de Branches (`feat/community`).
- Documentação técnica utilizando **Markdown**.

### 💻 Desafio Prático: Contribuição Open Source
A atividade consistiu em realizar uma contribuição no repositório oficial da Digital Innovation One (DIO), simulando um cenário real de colaboração em projetos abertos.

- **Repositório do Fork:** [dio-lab-open-source](https://github.com/lucianoarantesf/dio-lab-open-source)
- **Contribuição Realizada:** [Visualizar Pull Request / Branch de Funcionalidade](https://github.com/lucianoarantesf/dio-lab-open-source/tree/feat/community/lucianoarantesf)

> "A contribuição focou na documentação e apresentação formatada, reforçando a importância do README como porta de entrada para qualquer software de qualidade."
