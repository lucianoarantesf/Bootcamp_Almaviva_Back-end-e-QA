# 🚀 Jornada Java: Bootcamp Almaviva - Back-end & QA

Este repositório centraliza todos os meus estudos, projetos e desafios desenvolvidos durante o **Bootcamp Almaviva**, além de experimentações pessoais em Java Desktop e POO avançada.

## 🛠️ Stack Tecnológica & Ambiente
- **Linguagem:** Java 25 (OpenJDK)
- **Gestor de Versões:** [SDKMAN!](https://sdkman.io/)
- **IDE:** IntelliJ IDEA
- **Build Tool:** Maven (padrão da indústria para Spring Boot)
- **Sistema Operacional:** Linux (Ubuntu 24.04.4 LTS x86_64)

---

## 📂 Estrutura de Pastas (Baseada no Currículo)

Organização planejada para refletir a trilha de aprendizagem oficial:

- `01-onboarding-github/` - Fundamentos de colaboração e Git.
- `02-fundamentos-java/` - Sintaxe, tipos de dados e lógica básica.
- `03-orientacao-a-objetos/` - Classes, Métodos, Encapsulamento, Herança e Polimorfismo.
- `04-debug-excecoes-io/` - Tratamento de erros, Debugging e gestão de ficheiros.
- `05-collections-stream-api/` - Listas, Maps, Sets e manipulação de fluxos (Functional Java).
- `06-api-rest-spring-boot/` - Desenvolvimento de Back-end e APIs RESTful.
- `07-banco-de-dados/` - SQL e persistência de dados (JPA/Hibernate).
- `08-qualidade-de-software/` - Fundamentos de QA e processos de garantia de qualidade.
- `09-testes-software-junit/` - Testes unitários e automação com JUnit 5.
- `desafios-projeto/` - Repositório para os projetos práticos do Bootcamp.
- `treinos-pessoais/` - Projetos paralelos (Desktop Swing/JavaFX e lógica extra).

## ⚙️ Configuração para Execução
Como utilizo o **IntelliJ** e **SDKMAN!**, para rodar qualquer projeto deste repositório:
1. Verifique o JDK: `sdk current java` (deve ser v25).
2. No IntelliJ, certifique-se que o `Project SDK` está configurado corretamente em *File > Project Structure*.
3. Execute a classe principal através do ícone `Run` (Gutter icon).
